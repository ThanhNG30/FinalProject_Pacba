# FinalProject_Pacba
Final Project for Advanced Robotics course.
SET UP
1. Download the repository to your local folder: Click the green "Code" button, choose either open with Github Desktop or ZIP and open on VSCode
2. Create your own branch (if not yet done)  or
   navigate to your branch:
   git switch -c your-name
   git checkout your-name
4. Pull the starting code from main: git pull origin main
5. Add your changes locally and remotely: git add .
6. Commit your changes: git commit -m "add some message here"
7. Push your commits: git push -u origin your-name
8. Wait after discussion with other members, then merge your changes with main code
